# TryneX Frontend Deployment Guide (Netlify)

## Prerequisites
- Node.js (v18 or higher)
- Git
- GitHub account
- Netlify account

## Deployment Steps

### 1. Download and Setup
1. Download the generated TryneX files
2. Extract the files to a local folder (e.g., `trynex-frontend`)
3. Open terminal/command prompt in the project folder

### 2. Initialize Git Repository
```bash
git init
git add .
git commit -m "Initial TryneX frontend"
